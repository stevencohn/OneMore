Page Body Context Menu

Monday, February 20, 2023

7:02 AM

Right-click on text within a page to display its context menu. Chosen OneMore commands appear at the top of this menu as shown here. The commands that appear in this menu can be customized in the Settings dialog. By default, no OneMore commands are added; you can optionally select the ones you want to appear for quick and easy access.

 

 

 

In addition to the individual commands that you choose to appear in this menu, if you define search engines in the Settings dialog, they will also appear in this menu.

 

Container Outline Context Menu

The Page command Arrange Containers is available in the context menu that appears when right-clicking the container outline.
