
Set-ParagraphSpacing.ps1
Set the paragraph spacing for all paragraphs in each page of the current hierarchy.
There are parameters for spacing before, after, and in between paragraphs, as well
as parameters to control whether paragraphs of a list and inside tables are affected.

This is an example of embedded CSharp in a PowerShell script to make it easier
to navigate and manipulate XML.

--------------------------------------------------------------------------------

0. Save Set-ParagraphSpacing.ps1 to your local drive in some <path>
1. From the OneMore More menu, choose Run Plugin
2. Enter a name if you want to save this as a favorite such as "Set-ParagraphSpacing"
3. Enter the command: powershell
4. Enter arguments: -file <path>\Set-ParagraphSpacing.ps1
5. Set the Target field to the scope you want to affect
6. Click Save to save on the File/Plugins menu
7. Click Run to run now or activate from the File/Plugins menu
