
Export-PageInfo.ps1

Extracts reads the hierarchy and generates an Excel CSV file
containing information for every page within that scope.

--------------------------------------------------------------------------------

0. Save Export-PageInfo.ps1 to your local drive in some <path>
1. From the OneMore More menu, choose Run Plugin
2. Enter a name if you want to save this as a favorite such as "Export Page Info"
3. Enter the command: powershell
4. Enter arguments: -file <path>\Set-RainbowSections.ps1
5. Enter the full path of the .csv file that you want to create in User Arguments
6. Set the Target field to a scope other than Current Page
7. Click Save to save on the File/Plugins menu
8. Click Run to run now or activate from the File/Plugins menu
